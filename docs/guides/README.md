# 가이드 (Guides)

특정 작업을 수행하는 방법을 단계별로 설명하는 how-to 문서를 모은다. 프론트엔드 연동 가이드, 외부 연동 셋업, 운영 절차 등 "어떻게 하는가"에 답하는 문서가 여기에 속한다.

## 작성 규칙

- **파일명**: `yy-mm-dd-{슬러그}.md` (2자리 연도, 예: `26-06-01-fcm-push-setup.md`).
- **언어**: 한국어(코드/식별자만 영어).
- 가이드는 시점 스냅샷이다. 내용이 바뀌면 새 문서를 추가하거나 본문을 갱신하고 상단 날짜를 최신화한다.
