# 문제 해결 기록 (Troubleshooting)

장애·버그·삽질의 원인과 해결 과정을 기록한다. 같은 문제를 다시 만났을 때 빠르게 복구하고, 재발을 막기 위한 맥락을 남긴다.

## 작성 규칙

- **파일명**: `yy-mm-dd-{슬러그}.md` (2자리 연도, 예: `26-06-01-ddl-validate-failure.md`).
- **언어**: 한국어(코드/식별자만 영어).
- **구조**: ① 증상 → ② 원인 → ③ 해결 → ④ 재발 방지.
